## 🚀 Features
(1) Shortens long URLs  
(2) Supports custom short codes  
(3) QR code generation for links  
(4) Redis caching for faster access  
(5) Click tracking  

## 🛠️ Tech Stack
(1) ASP.NET Core : Used to build the backend API and handle requests  
(2) PostgreSQL : Used for storing URL mappings with indexing  
(3) Redis : Used for caching to reduce latency and improve performance  
(4) QRCoder : Used for generating QR codes for shortened URLs  
(5) JavaScript : Used for basic frontend interactions  

## ⚙️ How It Works
Takes long URL → generates short code → stores mapping → caches frequent requests → redirects efficiently.
